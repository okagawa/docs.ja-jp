# <a name="progressbar"></a>ProgressBar
<xref:System.Windows.Controls.ProgressBar> には、操作の進行状況が示されます。 <xref:System.Windows.Controls.ProgressBar> コントロールは、処理の進行状況に合わせてシステムの強調表示色で塗りつぶされるウィンドウで構成されます。
