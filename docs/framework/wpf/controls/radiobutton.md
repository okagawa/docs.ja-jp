# <a name="radiobutton"></a>RadioButton
<xref:System.Windows.Controls.RadioButton> コントロールは通常、グループにまとめられ、ユーザーはいくつかの選択肢から 1 つ選択できます。選択できるボタンは一度に 1 つだけです。
