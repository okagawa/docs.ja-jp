# <a name="tabcontrol"></a>TabControl
<xref:System.Windows.Controls.TabControl> 要素では、然るべきタブを選択した場合にアクセスできる、個別のページのコンテンツが表示されます。各タブには <xref:System.Windows.Controls.TabItem> が含まれます。
