# <a name="installation-guide"></a>インストール ガイド

.NET Framework は、さまざまなバージョンの Windows にインストールすることができます。

## <a name="supported-windows-versions"></a>サポートされている Windows バージョン

- [Windows 10 および Windows Server 2016](on-windows-10.md)
- [Windows 8.1 および Windows Server 2012 R2](on-windows-8-1.md)
- [Windows 8 と Windows Server 2012](on-windows-8.md)
- [Windows 7 および Windows Server 2008 R2](on-windows-7.md)
- [Windows Vista および Windows Server 2008](on-windows-vista.md)

## <a name="unsupported-windows-versions"></a>サポートされていない Windows バージョン

- [Windows XP および Windows Server 2003](on-windows-xp.md)
