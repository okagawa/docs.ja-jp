# <a name="icorprofilerthreadenumreset-method"></a>ICorProfilerThreadEnum::Reset メソッド

列挙子のカーソルをシーケンスの開始位置に移動します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT Reset();  
```
