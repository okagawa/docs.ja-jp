# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory メソッド
現在のスコープ内のすべてのメタデータを、指定したメモリ領域に保存します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT SaveToMemory (
    [out]  void        *pbData,
    [in]   ULONG       cbData
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `pbData`  
 入出力メタデータの書き込みを開始するアドレス。  
  
 `cbData`  
 から割り当てられたメモリのサイズ (バイト単位)。
