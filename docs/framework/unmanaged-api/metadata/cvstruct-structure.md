# <a name="cvstruct-structure"></a>CVStruct 構造体
モジュールまたは複合イメージをインストールするときに使用する情報が含まれます。  
  
## <a name="syntax"></a>構文  
  
```cpp  
typedef struct {  
    short Major;  
    short Minor;  
    short Sub;  
    short Build;  
} CVStruct;  
```  
  
## <a name="members"></a>メンバー  
  
|メンバー|説明|  
|------------|-----------------|  
|メジャー|メジャーバージョンのビルド番号。|  
|Minor|マイナーバージョンのビルド番号。|  
|Sub|サブビルド番号。|  
|Build|ビルド番号。|
