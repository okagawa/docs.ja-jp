# <a name="imetadatafilterunmarkall-method"></a>IMetaDataFilter::UnmarkAll メソッド
現在のメタデータスコープ内のすべてのトークンから処理マークを削除します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT UnmarkAll ();  
```  
  
## <a name="parameters"></a>パラメーター  
 `UnmarkAll`パラメーターを受け取りません。
