# <a name="imetadataemit2savedelta-method"></a>IMetaDataEmit2::SaveDelta メソッド
現在のエディットコンティニュセッションから、指定したファイルへの変更を保存します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT SaveDelta (  
    [in] LPCWSTR     szFile,
    [in] DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `szFile`  
 から変更を保存するファイル名。  
  
 `dwSaveFlags`  
 [in] 予約されています。 ゼロを指定してください。
