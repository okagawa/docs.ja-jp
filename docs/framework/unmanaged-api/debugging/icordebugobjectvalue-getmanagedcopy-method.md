# <a name="icordebugobjectvaluegetmanagedcopy-method"></a>ICorDebugObjectValue::GetManagedCopy メソッド

`GetManagedCopy` は互換性のために残されています。 このメソッドは呼び出さないでください。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetManagedCopy (  
    [out] IUnknown           **ppObject  
);  
```
