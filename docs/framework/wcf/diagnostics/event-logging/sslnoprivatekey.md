# <a name="sslnoprivatekey"></a>SslNoPrivateKey

Id: 154  
  
 重大度 : エラー  
  
 カテゴリ : TransactionBridge  
  
## <a name="description"></a>Description  

 このイベントは、特定のサブジェクト名と拇印を持つ ID 証明書に秘密キーがないことを示します。 イベントには、プロセス名とプロセス ID が表示されます。
