# <a name="service-calls-duration"></a>サービス: 呼び出しの期間

カウンター名: 呼び出しの期間  
  
## <a name="description"></a>Description  

 このサービスの呼び出しの平均期間です。 平均期間は、(N1-N0)/(D1-D0) という数式に基づいて計算されます。
