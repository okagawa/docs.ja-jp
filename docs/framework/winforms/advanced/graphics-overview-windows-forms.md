# <a name="graphics-overview-windows-forms"></a>グラフィックスの概要 (Windows フォーム)
GDI + は、プログラマがデバイスに依存しないアプリケーションを作成できるようにするグラフィックスデバイスインターフェイスです。 GDI + のサービスは、一連のマネージクラスを介して公開されます。
