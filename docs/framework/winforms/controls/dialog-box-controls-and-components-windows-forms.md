 [OpenFileDialog コンポーネントの概要](openfiledialog-component-overview-windows-forms.md)  
 ユーザーが構成済みのダイアログ ボックスを使用してファイルを開くことができるようにします。  
  
 [PageSetupDialog コンポーネントの概要](pagesetupdialog-component-overview-windows-forms.md)
