# <a name="creating-the-object-model"></a>オブジェクト モデルの作成
既存のデータベースからオブジェクト モデルを作成し、このオブジェクト モデルを既定の状態で使用できます。 オブジェクト モデルの多くの側面と動作をカスタマイズできます。  
  
 Visual Studio を使用している場合は、オブジェクト リレーショナル デザイナーを使用して、オブジェクト モデルを作成できます。
