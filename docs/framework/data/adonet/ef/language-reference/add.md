# <a name="-add"></a>+ (加算)
2 つの値を加算します。  
  
## <a name="syntax"></a>構文  
  
```csharp  
expression + expression  
```  
  
## <a name="arguments"></a>引数  
 `expression`  
 任意の数値データ型の有効な式。  
  
## <a name="result-types"></a>戻り値の型  
 2 つの引数の暗黙の型の昇格の結果であるデータ型。 暗黙の型の昇格について詳しくは、「[型システム](type-system-entity-sql.md)」をご覧ください。  
  
## <a name="remarks"></a>Remarks  
 EDM.String 型の場合は、値が連結されます。
