# <a name="expected--for-xml-end-tag"></a>XML 終了タグには '/' が必要です
XML リテラルの XML 終了タグに必要な '/' 文字がありません。  
  
 **エラー ID:** BC31167  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- XML リテラルの XML 終了タグの先頭に '/' 文字を追加します。
