# <a name="if-block-must-end-with-a-matching-end-if"></a>'#If' ブロックの最後には、対応する '#End If' が必要です
`#If` は条件付きコンパイル ディレクティブです。 `#If` ブロックが `#End If` ディレクティブで終了していません。  
  
 **エラー ID:** BC30012  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 条件付きコンパイル ブロックの最後に `#End If` ディレクティブ追加します。
