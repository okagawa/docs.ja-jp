# <a name="classes-can-inherit-only-from-other-classes"></a>クラスは、他のクラスからのみ継承できます
`Inherits` ステートメントで、クラス以外の型が基本型として指定されています。  
  
 **エラー ID:** BC30258  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 基本型を既に定義されているクラスに変更します。
