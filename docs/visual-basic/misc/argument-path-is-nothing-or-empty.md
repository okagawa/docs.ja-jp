# <a name="argument-path-is-nothing-or-empty"></a>引数 'Path' は Nothing か、または空です
パス名は、 `Nothing` に初期化されるか空であるため、使用できません。 パスには、特定のデータ型と正しい名前を設定する必要があります。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 正しいパス名を追加します。
