# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>'Continue' ステートメントは、イミディエイト ウィンドウでは有効ではありません
' Continue ' ステートメントは、ソースコードでのみ使用できます。  
  
 **エラー ID:** BC30780  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- デバッグコードから ' Continue ' ステートメントを削除します。
