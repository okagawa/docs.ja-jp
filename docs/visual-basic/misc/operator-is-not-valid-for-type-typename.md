# <a name="operator-is-not-valid-for-type-typename"></a>演算子は型 ' ' に対して有効ではありません。 \<typename>
演算子は、型に適切ではありません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 演算子が演算対象の型に適していることを確認します。
