# <a name="the-file-is-currently-open-for-writing"></a>ファイルは現在書き込み用に開かれています
現在書き込み用に開かれているファイルにアクセスしようとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- ファイルを閉じて、操作をやり直します。
