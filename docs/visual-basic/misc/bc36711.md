# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>パラメーター ' ' と一致する引数 \<parametername> は ' ' に縮小変換します。 \<typename>
オーバーロードされたメソッドを呼び出しましたが、コンパイラは縮小変換せずに呼び出すことができるメソッドを検出できません。 *縮小変換* により、有効値の一部を保持できない可能性のあるデータ型に値が変更されます。
