# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>この式の評価中に 0 で除算しました
この式を評価する際に、数値を 0 で除算しようとしました。  
  
 **エラー ID:** BC30542  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 定数の値と変数の値を確認し、式の分母に 0 がどのようにして発生したかを判断します。
