# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>引数 '\<argumentname>' の型は '\<typename>' です。数値ではありません。
数値引数が期待される状況で数値以外の引数が指定されています。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 引数が数値であることを確認します。
