# <a name="comma-expected"></a>コンマが必要です。
引数リストの 2 つの引数の間にコンマ (,) がありません。  
  
 **エラー ID:** BC30196  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- すべての引数がコンマで区切られていることを確認します。
