# <a name="next-expected"></a>'Next' が必要です

終端の `On Error Resume` がない状態で `Next`コンストラクトが指定されています。  
  
 **エラー ID:** BC30003  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- `Next` ステートメントの末尾に `On Error` キーワードを追加します。
