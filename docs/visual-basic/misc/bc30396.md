# <a name="specifier-is-not-valid-on-an-enum-declaration"></a>' \<specifier> ' は列挙型宣言では有効ではありません。
`Enum` 宣言で無効な指定子が使用されました。  
  
 **エラー ID:** BC30396  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 指定子を削除します。
