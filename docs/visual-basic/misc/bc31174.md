# <a name="xml-processing-instruction-name-name-is-not-valid"></a>XML 処理命令の名前 'name' が有効ではありません
XML 処理命令に指定した名前が正しくありません。  
  
 **エラー ID:** BC31174  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- XML 処理命令の名前のエラーを修正するか、または削除します。
