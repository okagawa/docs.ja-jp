# <a name="expected-closing--for-xml-processor-instruction"></a>XML プロセッサ命令の終わりに '? > ' が必要です
XML 処理命令リテラルの終わりに必要な `?>` 文字がありません。  
  
 **エラー ID:** BC31160  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- XML 処理命令リテラルの終わりに必要な `?>` 文字を追加します。
