# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>集計関数名で型文字を使用することはできません
型文字が集計関数の名前の末尾に含まれています。  
  
 **エラー ID:** BC36617  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 集計関数名から型文字を削除します。
