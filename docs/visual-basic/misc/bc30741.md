# <a name="typename1-cannot-be-converted-to-typename2"></a>' ' \<typename1> を ' ' に変換することはできません。 \<typename2>
2 つの互換性のない型の間で型変換が試行されました。  
  
 **エラー ID:** BC30741  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- キャストされている型を確認します。
