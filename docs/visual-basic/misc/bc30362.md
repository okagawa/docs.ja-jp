# <a name="propertyname1-cannot-override-propertyname2-because-they-differ-by-readonly-or-writeonly"></a>' \<propertyname1> \<propertyname2> ReadOnly ' または ' WriteOnly ' が異なるため、' ' で ' ' をオーバーライドすることはできません。
1 番目のプロパティを、 `ReadOnly` か `WriteOnly`かの違いしかない 2 番目のプロパティでオーバーライドしようとしました。  
  
 **エラー ID:** BC30362  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- プロパティどうしの間に `ReadOnly` と `WriteOnly`以外の違いがあるようにします。
