# <a name="keyword-does-not-name-a-type"></a>キーワードは型の名前を指定しません
型名を指定する必要がある場所にキーワードが指定されています。 このコンテキストでは、基本データ型、または型を表す宣言された要素名のいずれかが呼び出されます。  
  
 **エラー ID:** BC30180  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- キーワードを有効な型名に置き換えます。
