# <a name="on-visual-basic"></a>On (Visual Basic)
実行時エラーに対する応答を導入するか、コンパイラオプションをオンにします。  
  
 クエリでは、は、結合されているコレクションのキーを識別します。

 キーワードは、次の `On` コンテキストで使用されます。

 [Join 句](../language-reference/queries/join-clause.md)
