# <a name="operator-is-not-valid-for-name1-and-name2"></a>演算子は ' \<name1> ' および ' ' に対して有効ではありません。 \<name2>
演算子は演算対象の型に適していません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 演算子が演算対象の型に適していることを確認します。
