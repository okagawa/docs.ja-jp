# <a name="device-io-error"></a>デバイス I/O エラーです。
プログラムがプリンターやディスク ドライブなどのデバイスを使用しているときに、入力または出力のエラーが発生しました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- デバイスが適切に動作していることを確認してから、操作をやり直してください。
