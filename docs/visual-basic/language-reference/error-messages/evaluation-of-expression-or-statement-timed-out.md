# <a name="evaluation-of-expression-or-statement-timed-out"></a>式またはステートメントの評価がタイムアウトしました。
式の評価が時間内に完了しませんでした。  
  
 **エラー ID:** BC30722  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 入力したコードが正しいことを確認してください。  
  
2. 実行時間が短くなるように式を単純化してください。
