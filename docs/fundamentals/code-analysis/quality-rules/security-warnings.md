# <a name="security-rules"></a>セキュリティ規則

セキュリティ規則は、より安全なライブラリとアプリケーションをサポートします。 これらの規則は、プログラムのセキュリティの欠陥を防ぐのに役立ちます。 これらの規則のいずれかを無効にする場合は、コードで理由を明確にマークし、開発プロジェクトの指定されたセキュリティ担当者にも通知する必要があります。
