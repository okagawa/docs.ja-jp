# <a name="default-c-reference"></a>default (C# リファレンス)

`default` キーワードは、以下の 2 とおりの方法で使用できます。

- [`switch` ステートメント](switch.md)で既定のラベルを指定するため。
- 型の既定値を生成するための[既定の演算子またはリテラル](../operators/default.md)として。
