# <a name="compiler-warning-level-1-cs0602"></a>コンパイラの警告 (レベル 1) CS0602

機能 'old_feature' は非推奨とされます。 'new_feature' を使用してください。

 コードで使用される言語機能 (*old_feature*) は引き続きサポートされていますが、将来のリリースでサポートが削除される可能性があります。 代わりに、構文 (*new_feature*) を使用することをお勧めします。

> [!NOTE]
> このコンパイラ警告は Roslyn では使用されなくなりました。
