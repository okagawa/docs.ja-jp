# <a name="compiler-error-cs0543"></a>コンパイラ エラー CS0543
'enumeration': 列挙子の値は型に対して大きすぎます  
  
 [列挙](../language-reference/builtin-types/enum.md) 内の要素に割り当てられた値がデータ型の範囲外です。  
  
 次の例では CS0543 が生成されます。  
  
```csharp  
// CS0543.cs  
namespace x  
{  
   enum I : byte  
   {a = 255, b, c}   // CS0543  
   public class clx  
   {  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
